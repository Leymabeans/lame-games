About

Welcome to the Lame Games website! Lame games is an organization created to bring programming and fun together. The creators, Nathan Ley and Andrew Zweifel, both enjoy game development and competition so this series was brought to life.


The competition is a timed event in which both Nathan and Andrew must create the best game possible using the programming language, Python.The reason behind the name "Lame Games", is due to the fact that the games that are created in a rush don't usually end up well.


Lame Games is creating new content, whether that be games, social media, videos, or tutorials. We are alwasy accepting new members to be competitors in both our game development and game testing sections. Feel free to contact us if you are interested.

Rules

These are the rules that Nathan and Andrew must follow when making a game.


  1. The time limit in the competition is 30 minutes.
  2. Both must start with a clean, empty file.
  3. The only language used must be Python.
  4. No copying and pasting from previous projects is allowed
